## Configuration

### Requisites

1. PHP 7.3^
2. Composer 1.9^

### Installation

After forking/cloning you can follow those steps to run the project:

To update the dependencies:

```sh
composer install
```

Update the `config\app_local.php` with your database information

To start the server:

```sh
bin\cake server
```
